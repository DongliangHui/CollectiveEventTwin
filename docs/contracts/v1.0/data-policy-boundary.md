# 禁用数据策略 v1.0

状态：冻结版

## 禁止进入生产链路

- cookie 池。
- CAPTCHA 绕过。
- 私信/私域聊天采集。
- 未授权登录态采集。
- 冒充真人互动。
- 绕平台访问控制。
- 非法个人敏感信息采集。
- 无授权的账号池、多账号轮换。
- Agent 假装真人用户进行互动。

## 阻断行为

被策略阻断的数据源必须：

- 返回 `SOURCE_POLICY_BLOCKED`。
- 写 `source_policies`。
- 写 `audit_logs`。
- 不创建 `collection_runs`。
- 在数据源健康和页面 `degraded_sources` 中可见。

## 允许但需约束

| 类型 | 条件 |
| --- | --- |
| 公开网页 | 遵守公开访问边界和 robots/频率限制 |
| 官方 API | 使用授权 key，记录限流和失败 |
| 手工上传 | 上传者负责授权声明，系统保留来源 |
| 合成数据 | 全链路 synthetic 标记 |
| 媒体文件 | 脱敏后才能进入客户可见事实 |

## Review 阻断

任何数据源、算法输出、媒体输出或报告若无法证明来源边界，第三方检查必须 FAIL。
