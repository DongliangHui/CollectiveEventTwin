# Investor Demo - Risk Intelligence Twin

This local Figma development plugin generates the investor demo prototype in the currently open Figma file.

## How to run

1. Open the target Figma file.
2. In Figma, go to `Plugins -> Development -> Import plugin from manifest...`.
3. Select `figma-investor-demo-plugin/manifest.json`.
4. Run `Plugins -> Development -> Investor Demo - Risk Intelligence Twin`.

The plugin creates a page named `Investor Demo - Risk Intelligence Twin` with 8 linked frames:

- `00 Narrative Brief`
- `01 Risk Command Workspace`
- `02 Event & Evidence Chain`
- `03 Scenario Sandbox`
- `04 Agent Council`
- `05 Investment Lens`
- `06 Decision Report`
- `07 Transferability`

It uses mock data derived from the Obsidian materials and includes prototype links for the main investor story path.
